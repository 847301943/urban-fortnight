# 大A智能选股：无 Token 自动更新版

这是一个无需 Tushare Token、无需自建服务器的静态选股网页。GitHub Actions 在后台获取公开数据并生成 `data/stocks.json`，手机打开网页时只读取这一份文件。

## 一键部署

1. 在 GitHub 新建一个 **Public** 仓库。
2. 解压本压缩包，把全部文件上传到仓库根目录。必须保留隐藏目录 `.github/workflows/`。
3. 进入仓库 `Settings → Pages → Build and deployment → Source`，选择 **GitHub Actions**。
4. 进入 `Actions → 更新并发布大A智能选股网页 → Run workflow`。
5. 等待绿色对勾，任务页面会显示网页地址。

以后工作日收盘后会自动更新，也可以随时手动运行。

## 新增财务质量指标

### 全A股自动覆盖

- ROE、毛利率、净利率、资产负债率；
- 经营现金流/净利润；
- ROIC；
- 自由现金流及自由现金流收益率；
- 最近3个完整年度的经营现金流、自由现金流；
- 近3年现金流为正的年数与趋势；
- 营收和净利润3年复合增速。

### 两层计算口径

1. **精确候选**：后台会对初步质量排名靠前的候选读取完整年度三张财务报表。
   - 自由现金流 = 经营现金流 − 资本开支；
   - ROIC = 税后营业利润 ÷（股东权益 + 有息负债 − 现金）。
2. **全市场估算**：未进入精算队列的非金融企业仍会计算指标。
   - ROIC采用税后营业利润/经营投入资本估算；
   - 自由现金流采用“经营现金流 + 投资现金流净额”作为保守代理。

网页会逐股显示“精确、部分精确、估算或不适用”，不会把代理口径伪装成精确值。

银行、保险、证券等金融企业不适合直接套用工业企业ROIC和自由现金流公式，因此相关字段会标记为“不适用”。

## 网页筛选项

- 最低ROE、最低ROIC；
- 最低自由现金流收益率；
- 近3年经营现金流至少为正多少年；
- 只看精确ROIC/自由现金流候选；
- 负债率、PE、市值、利润增速和趋势条件。

## 调整精算股票数量

工作流默认精算前160只初步质量候选。可编辑：

```yaml
DETAIL_LIMIT: "160"
DETAIL_WORKERS: "5"
```

数量越大，覆盖越多，但运行更慢，也更容易受到公开接口限流。建议先保持默认。

## 数据风险

公开数据接口可能延迟、缺失或调整。更新失败时，GitHub Pages 会继续保留上一次成功发布的数据。评分只用于缩小研究范围，不构成投资建议；财报原文、会计口径、行业特性、护城河和管理层仍需人工核查。
