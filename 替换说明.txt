本升级包是“隔日短线专版”，必须同时替换/新增以下文件：

仓库根目录：
- index.html
- requirements.txt
- README.md

scripts 目录：
- scripts/update_data.py
- scripts/short_strategy.py
- scripts/backtest_short.py

.github/workflows 目录：
- .github/workflows/update-stock-data.yml

上传后：
1. Settings → Pages → Source 确认是 GitHub Actions。
2. Actions → 更新并发布大A智能选股网页 → Run workflow。
3. 首次运行最多刷新1200只隔日候选，可能比旧版耗时更长。
4. 成功后查看网页“策略回测与数据审计”。

注意：
- 不是轮换池，入选当日隔日池的股票每次运行都会刷新。
- 未进入隔日池不等于数据缺失。
- 收盘后生成信号，次日高开超过2.5%不追。
- A股T+1，次日买入后最早再下一交易日退出。
